举例：左键单击“生物”分类中的“猎犬”生成1只猎犬，右键单击“猎犬”生成10只猎犬。 [*] 按住[b]Ctrl键[/b]并 单击鼠标左键 - 将该项目添加到自定义分类中 [*] 如果项目已存在于自定义分类中，再次按住[b]Ctrl键[/b]并单击鼠标左键可以从自定义分类中移除 [*] 举例：按住Ctrl键并左键单击“装备”分类中的“长矛”、“猪皮帽”、“木甲”，将这3件装备添加到自定义分类中。 [*] 再次按住Ctrl键并左键单击“自定义”分类中的“长矛”、“猪皮帽”、“木甲”，将这3件装备从自定义分类中移除。 [*] 按住[b]Shift键[/b]并 单击鼠标左键 - 获取1份该项目的制作材料（和左键生成的数量相同） [*] 按住[b]Shift键[/b]并 单击鼠标右键 - 获取10份该项目的制作材料（和右键生成的数量相同） [*] 举例：按住[b]Shift键[/b]并左键单击“建筑”分类中的“烹煮锅”，获得1份烹煮锅的制作材料（石砖块*3，木炭*6，树枝*6） [*] 按住[b]Shift键[/b]并右键单击“建筑”分类中的“烹煮锅”，获得10份烹煮锅的制作材料（石砖块*30，木炭*60，树枝*60） [*] 按住 [b]Ctrl键[/b]和[b]Alt键[/b]并 单击鼠标右键 - 传送到该项目的位置（如果该项目存在） [*] 举例：按住[b]Ctrl键[/b]和[b]Alt键[/b]并右键单击“巢穴”分类中的“猪人房”，可以传送到当前地图的猪人房位置。 [*] 再次执行同样的操作，可以传送到当前地图的下一个猪人房位置。 [*] *食物、工具、武器、护甲等可携带项目会直接添加到你的物品栏或背包中。 [*] *建筑、生物、植物、矿物等非可携带项目会生成在玩家当前位置。 [*][h1]属性调整：[/h1] [*] 点击潮湿度按钮，将潮湿度调整至0% [*] 按住[b]Ctrl键[/b]并点击潮湿度按钮，将潮湿度调整至100% [*] 点击温度按钮，将温度调整至25℃ [*] 按住[b]Ctrl键[/b]并点击温度按钮，将温度调整为当前环境温度。 [*] 点击3种变身按钮，将变身值调整至100%并变为指定形态（仅限伍迪） [*] 处于变身形态下，点击任意的变身按钮，将变身值再次调整至100%（仅限伍迪）
